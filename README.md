## 🔍 About catalyst-n1

Catalyst N1 is an open-source neuromorphic processor modeled after Intel’s Loihi 1. It features 128 cores and supports 131,000 neurons. This processor uses a specialized 14-opcode learning instruction set architecture (ISA) tested on FPGA hardware via AWS F2 instances. The software brings cutting-edge neural computing concepts closer to everyday use.  

This project is designed for users interested in experimenting with or learning about neuromorphic computing. You do not need programming experience to set up or run the software on your Windows PC. The instructions below will guide you step-by-step to download and run catalyst-n1 smoothly.
